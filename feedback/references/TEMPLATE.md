# Feedback Templates

## Bug Report Template

```markdown
# Bug Report: [Brief Title]

**Type**: Bug Report
**Date**: [YYYY-MM-DD]
**Submitted by**: [User]
**Severity**: [Low | Medium | High | Critical]

## Description
[Clear description of what's broken or not working]

## Steps to Reproduce
1. [First step]
2. [Second step]
3. [Third step]

## Expected Behavior
[What should happen]

## Actual Behavior
[What actually happens]

## Environment
- **OS**: [e.g., Windows 11, macOS 14.0, Ubuntu 22.04]
- **Browser**: [e.g., Chrome 120, Firefox 121]
- **Version**: [Application version]
- **Device**: [Desktop, Mobile, Tablet]

## Screenshots/Logs
[Attach or describe any relevant screenshots or error logs]

## Additional Context
[Any other relevant information]

## Status
- [x] Submitted
- [ ] Under Review
- [ ] In Progress
- [ ] Completed
```
